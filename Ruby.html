Ruby

This page has details about uses, and the history and evolution of Ruby.

What is Ruby Used For?

Ruby is mostly used for creating web applications, but can be used for many other things too.

General History and Evolution of Ruby.

Ruby was created around the mid 1990s by Yukihiro Matsumoto, a Japanese computer programmer, and released in 1995. When creating Ruby, Matsumoto tried to combine elements of other languages, including Perl, Eiffel, Smalltalk, Lisp, and Ada. Ruby is mostly famous for its "Ruby on Rails" (ROR) framework, which released in 2005.

ROR 1.0

Ruby on Rails revolutionized how web development was done, and added:

  • Duck typing
  • WEBrick
  • StringIO
  • PP
  • Fully qualified names
  • Native YAML support
  • Open-uri

ROR 2.0

Released in December 2007, and included:

  • More speed improvements
  • Addition of built in syntax documentation
  • Keyword arguments
  • Optimized garbage collection
  • UTF-8 by default
  • Some refinements

ROR 3.0

Released in August 2010, and included:

  • New router with emphasis on RESTful declarations
  • New action mailer API
  • New chainable query language built on relational algebra

ROR 4.0

Released in June 2013, and included:

  • Russian Doll Caching
  • Turbolinks
  • Strong Parameters

ROR 5.0

Released June 2016, and included:

  • Action Cable
  • Test Runner
  • Rails API
  • Sprockets 3
  • Active record attributes API
  • Exclusive use of rails CLI over Rake
  • Turbolinks 5

Companies that use Ruby
