PHP

This page has details about uses, and the history and evolution of PHP.

What is PHP Used For?

PHP is a general purpose programming language, so it can be used to do about anything. PHP is mainly used for three things:

  1. Server-side scripting
  2. Command line scripting
  3. Writing desktop applications

General History and Evolution of PHP.

PHP was created in 1994 by Rasmus Lerdorf. It was originally used to track visits to Lerdorf’s online resume, and he named it “Personal Home Page” tools (PHP tools). the first version was a simple set of Common Gateway Interface (CGI) binaries written in C, but Lerdorf eventually developed a new model that had more implementation, like database interaction, and was used to develop simple dynamic web applications. In 1995, Lerdorf released the source code to the public.

FI

In September, Lerdorf actually dropped the PHP name for a bit, calling it FI (forms interpreter). This new implementation added:

  • Perl-like variables
  • Automatic interpretation of form variables
  • HTML embedded syntax

In October, Lerdorf released a complete rewrite of the language. The PHP name was brought back, and this release was considered more of an advanced scripting interface. The new version purposefully resembled C in structure, making it easier for developers to adopt.

PHP 2

The code got completely redone in April 1996, being now named PHP/FI, or PHP 2. This new generation really began to change PHP from a set of tools to an outright programming language. PHP 2 included:

  • Support for DBM
  • mSQL
  • Postgres95
  • Cookies
  • User-defined function support

PHP 3

In 1997, Andi Gutmans and Zeev Suraski began to work with Lerdorf to create what would become PHP 3. PHP 3 included:

  • Mature interface for multiple databases
  • Protocols
  • API

PHP 3 was beginning to be hosted on Windows 95, 98, and NT, and Macintosh operating systems.

PHP 4

PHP 4 was released in 1999, and included:

  • A rewritten core
  • Support for more web servers
  • HTTP sessions
  • Output buffering
  • More secure ways of handling user input
  • Several new language constructs

PHP 5

PHP 5 was released in July 2004, which was mainly driven by its Zend Engine 2.0.
